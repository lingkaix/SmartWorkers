# temp/ (scratch)

This folder is scratch space for intermediate files created while working on tasks in this repo.

## Rules

- Anything here may be deleted at any time.
- Do not place final deliverables here; final outputs go in `artifacts/`.
- Create a per-task subfolder for each run.
- Each per-task subfolder must include a short `README.md` describing progress and listing files.

## Safety

- Avoid copying secrets into files here.
- If a task requires referencing config, use workspace-root `workers.jsonc` and never write secret values into logs or readmes.
