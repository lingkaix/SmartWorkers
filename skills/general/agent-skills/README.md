# Agent-Skills Helper Pack

This folder is the shared SmartWorkers authoring kit.

It is not the installed maintainer skill. The installed skill is:

- `skills/general/smart-skill-maker/`

Use this helper pack for the pieces that `smart-skill-maker` relies on:

- `assets/templates/SKILL.md` — default SmartWorkers skill scaffold
- `references/AUTHORING.md` — canonical SmartWorkers authoring guide
- `scripts/new_skill.sh` — safe scaffolder for new source skills

Keep these files aligned with `skills/general/smart-skill-maker/` whenever the repo's skill-authoring conventions change.
