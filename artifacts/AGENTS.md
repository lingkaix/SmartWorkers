# artifacts/ (deliverables)

This folder stores final deliverables produced by tasks in this repo.

## Rules

- Create a per-task subfolder for each run.
- Each per-task subfolder must include a short `README.md` describing what was produced and listing files.
- Intermediate or messy working files belong in `temp/`, not here.

## Safety

- Do not include secrets in deliverables.
- If a deliverable must mention config, reference key names/paths (for example: `openai.apiKey`) but never include values.
